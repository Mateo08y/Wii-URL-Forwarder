# Wii URL Forwarder Base

Este repositorio contiene el primer **WAD base editable** para crear canales de Wii que abren una **URL personalizada** en el Navegador de Internet de la consola.

La comunidad nunca tuvo un proyecto organizado para esto, así que este repositorio sirve como punto de partida para cualquiera que quiera crear sus propios canales web.

---

## 🎯 ¿Qué es un URL Forwarder?

Un **URL Forwarder** es un canal de Wii que, al abrirse, lanza el Navegador de Internet y carga automáticamente una página web específica (por ejemplo: minecraft.net, tlauncher.org, YouTube, etc.).

Este repositorio incluye:

- Un **WAD base** editable con CustomizeMii  
- Un **DOL personalizado** que abre una URL  
- Archivos listos para modificar y compilar  
- Instrucciones claras para crear tu propio canal

---

## 🛠 Requisitos

Para editar este WAD necesitas:

- **CustomizeMii** (Windows)
- **Un PC** (Windows recomendado)
- El archivo `boot.dol` incluido en este repositorio
- El WAD base incluido

---

## 🧩 Cómo cambiar la URL

1. Abre **CustomizeMii** en tu PC.  
2. Carga el archivo `Wii_URL_Forwarder_Base.wad`.  
3. Ve a la pestaña **DOL**.  
4. Reemplaza el `boot.dol` por el que quieras usar.  
5. Si quieres cambiar la URL, edita el archivo `url.txt` antes de compilar.  
6. Guarda el nuevo WAD con **Create WAD**.

---

## 📥 Instalación en la Wii

1. Copia el WAD final a tu tarjeta SD.  
2. Abre **Wii Mod Lite** o **YAWMM**.  
3. Instala el WAD.  
4. El canal aparecerá en el menú de Wii.
